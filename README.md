# Weather Dashboard

A real-time weather forecast app using the free Open-Meteo API — no API key required.

## Features
- Search any city worldwide
- Current temperature, feels-like, humidity, wind speed, visibility
- 7-day forecast with daily min/max and precipitation
- Background gradient changes based on weather condition
- Powered by Open-Meteo (free, no key needed) + Open-Meteo Geocoding

## Tech Stack
React 18 · TypeScript · Tailwind CSS v4 · Vite · Open-Meteo API

## Getting Started
```bash
npm install
npm run dev
```
No environment variables needed — Open-Meteo is completely free and open.
